# 🏠 PropWise India — Real Estate Intelligence Platform

Full-stack ML web app for Indian real estate price prediction and market analytics.

## Features
- **AI Price Prediction** — 34 cities, 100+ localities, property type, floor, furnishing
- **Market Dashboard** — City stats, avg price/sqft charts, full data table
- **City Comparison** — Compare up to 5 cities side by side with charts
- **Market Heatmap** — Visual price intensity across all cities
- **Price Trends** — Quarterly price movement 2020–2024
- **ROI Calculator** — Capital gains + rental income + EMI analysis
- **News Feed** — Live real estate headlines
- **Model Insights** — EDA plots, feature importance, interview Q&A

## Quick Start
```bash
pip install -r requirements.txt
python backend/data/generate_dataset.py
python backend/train_model.py
python backend/app.py
# → http://localhost:5001
```

## Tech Stack
Python · Pandas · Scikit-learn · Flask · Chart.js · HTML/CSS/JS
